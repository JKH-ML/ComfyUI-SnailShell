# ComfyUI-SnailShell 🐌🐚

A specialized steganography custom node for ComfyUI. Hide your secret data (Snail) inside a generated carrier image (Shell).

## Features
- **SnailEncoder (v2.7)**: Hide a single image or a batch of images (video) inside a snail shell image.
- **SnailDecoder (v2.7)**: Extract the hidden snail (Image/Video) from the shell.
- **Zero-Config**: Automatically determines the best bit-depth and shell size.
- **Password Protection**: Secure your hidden data with a password.

## Installation
1. Navigate to your `ComfyUI/custom_nodes` folder.
2. `git clone https://github.com/YOUR_USERNAME/ComfyUI-SnailShell.git`
3. Restart ComfyUI.

## Browser Support
Use the **Flip the Shell** Chrome extension to reveal snails directly on the web!
